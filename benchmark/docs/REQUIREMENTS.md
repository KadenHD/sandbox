# Requirements

Hereâ€™s a breakdown of the installation requirements for each language.

## C

Install [gcc](https://gcc.gnu.org/install/download.html)

Verify:

```bash
gcc -v
```

## C++

Install [g++](https://gcc.gnu.org/install/download.html)

Verify:

```bash
g++ -v
```

## C\#

Install [csc](https://learn.microsoft.com/en-us/answers/questions/1138661/how-can-i-use-csc-exe-(-net-framework-executable))

Verify:

```bash
csc -help
```

## Go

Install [Go](https://go.dev/doc/install)

Verify:

```bash
go version
```

## Java

Install [Oracle JDK](https://www.oracle.com/java/technologies/downloads/)

Verify:

```bash
java -version
javac -version
```

## JavaScript

Install [Node.js](https://nodejs.org/en/download/)

Verify:

```bash
node -v
```

## PHP

Install [PHP runtime](https://www.php.net/downloads.php)

Verify:

```bash
php -v
```

## Python

Install [Python interpreter](https://www.python.org/downloads/)

Verify:

```bash
python --version
```

## Ruby

Install [Ruby+Devkit](https://www.ruby-lang.org/fr/documentation/installation/)

Verify:

```bash
ruby -v
```

## Rust

Install [Rust toolchain (compiler + Cargo)](https://rustup.rs/)

Verify:

```bash
rustc --version
cargo --version
```

## TypeScript

Install [Node.js](https://nodejs.org/en/download/) and :

```bash
npm install -g typescript
```

Verify:

```bash
tsc -v
node -v
```
